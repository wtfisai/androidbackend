Important Security Notice
Remote control features require special Android permissions. These permissions allow full device control and should only be granted on trusted devices.
How to Enable Permissions:

  1. Enable Developer Options: Settings → About → Tap "Build Number" 7 times
  2. Enable USB Debugging: Settings → Developer Options → USB Debugging
  3. Connect device to computer via USB
  4. On computer: adb devices (accept prompt on phone)
  5. Grant permissions: adb shell pm grant com.termux android.permission.INJECT_EVENTS
